This is a project containing two html files, one for a webpage and one for a contact form. 
The webpage's file is the index.html file. This file contains a header section, main section and footer section for a hub's webpage. 
The header section contains the name of the webpage and a navigation bar containing internal links to different parts of the webpage and an external link connecting this file to the contact form html file. 
The main section contains setion tags grouping different contents in the webpage, with comments above all for description.
Each section contains differentbelemets and tags for its content, including article tags, heading tags, list tags, aside tags, div tags to break out contents, media elemets, iframe tags, etc.
The footer section contains the copyright info of the webpage.
The second html file is the contact form, named contact.html.
Using the form tag, this webpage contains form options to be filled by a user.
The form consists of label tags, input types for name, email, password, phone number as an extra item, radio and checkbox for options, dropdown select tag, a textarea and submit button.
The form made use of several attributes as well.
It further consists of a link to go back to the hub webpage after submission.
